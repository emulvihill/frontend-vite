WebGL MolView provides information

WebGL MolView measures distances

Select any two atoms to see the distance in Angstroms between them.

  • Learn More

WebGL MolView measures rotation angles

Select three adjacent atoms to determine the rotation angle between the two bonds connecting them.

  • Learn More

WebGL MolView measures torsion angles

Select four adjacent atoms to determine the torsion angle between the central bond and the two peripheral bonds.

WebGL MolView is open source and free to use

WebGL MolView is an open-source project that is free to use. It is built using the Three.js library.
